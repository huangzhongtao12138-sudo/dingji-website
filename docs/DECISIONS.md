# 鼎基装饰官网长期决策记录

本文件记录会影响未来维护的重要选择。新决策按日期追加，不覆盖旧记录；若决策被替代，应注明替代关系。

## 2026-07-22：项目记忆写入仓库

- 决策：项目知识以 `AGENTS.md`、`docs/PROJECT_MEMORY.md` 和本文件作为新对话的长期上下文。
- 原因：账号级隐藏记忆和旧聊天记录不能保证在所有新对话中自动可用，项目文件更稳定、可审查、可同步。
- 影响：公司资料、网站结构、技术部署或重要维护规则发生变化时，必须同步更新项目资料库。
- 安全：项目记忆不得保存密码、令牌、验证码或其他敏感信息。

## 2026-07-22：搜索站点子链接结构

- 决策：首页保留清晰的独立入口，并为“联系我们、咨询服务、连锁店装修、常见问题”建立可收录页面。
- 原因：旧官网搜索结果曾展示多个站点子链接；新官网需要更清晰的站点层级和内部链接。
- 影响：这些页面属于当前重点搜索入口，不能无故改回首页锚点或隐藏在脚本交互中。
- 边界：Google 站点子链接自动生成，网站不能保证展示。

## 2026-07-22：生产部署方式

- 决策：继续使用 GitHub `main` → Vercel 自动部署。
- 生产目录：`outputs/`
- 生产首页：`outputs/index.html`
- 影响：推送 `main` 会影响正式官网；推送前需完成验证并明确告知用户。

## 2026-07-22：全站滚动统一使用 Lenis

- 决策：生产页面统一使用本地托管的 Lenis 1.3.25，替代分散的 CSS 原生平滑滚动和首页分区滚动动画。
- 原因：统一桌面端滚轮惯性、锚点跳转与滚动节奏，减少不同页面之间的体验差异。
- 边界：手机触控保持原生滚动；用户启用“减少动态效果”时不初始化 Lenis；横向案例区域和弹窗不由 Lenis 接管。
- 维护：依赖固定保存在 `outputs/site-assets/`，升级版本前需重新验证首页、内页、锚点、横向图库与 390px 手机端。

## 2026-07-22：关于页采用渐进式 3D 视觉

- 决策：关于页保持纯静态 HTML/CSS/JavaScript 架构，不为单页引入 React、Tailwind 或组件框架；桌面端按需加载 Spline Viewer 和本地场景文件。
- 原因：吸收用户提供的动态标题、能力切换和 3D 视觉方案，同时避免改变整站技术栈或让移动端承担较重的 3D 资源。
- 边界：移动端、省流量模式、减少动态效果或外部查看器加载失败时，必须保留完整内容和轻量 CSS 视觉回退；核心信息与导航不能依赖 3D 场景。
- 展示要求：3D 只作为与首屏融合的背景动效，不使用液态玻璃卡片承载，不展示场景自带英文文字、编号或坐标；关于页导航必须沿用官网统一的白色液态导航。
- 屏幕适配：首屏同时按屏幕宽度和可用高度适配，常见 720–900px 高度的桌面屏幕不得裁切核心文案、操作入口或数据栏。
- 内容：关于页继续使用用户确认的 1998 年、26,000㎡生产基地、3,000+项目实践、80+服务城市与合作伙伴等企业数据。

## 既有设计方向

- 保留红色品牌识别、米灰纸张背景、玻璃质感卡片、粗体中文标题和圆角导航。
- 新页面延续现有体系，不因单次需求重做整站。
- 桌面端与手机端同等重要，手机端不得出现非预期横向溢出。
